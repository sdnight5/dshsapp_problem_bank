# 문제
두 정수 A와 B를 입력받은 다음, A/B를 출력하는 프로그램을 작성하시오.
# 입력
```
A B
```
- 첫째 줄에 A와 B가 주어진다. 
- $0 < A, B < 10$
# 출력
첫째 줄에 A/B를 소숫점 아래 9자리까지 반올림해서 출력한다.
# 예제
## 예제 1
### 입력
```
1 3
```
### 출력
```
0.333333333
```
## 예제 2
### 입력
```
4 5
```
### 출력
```
0.800000000
```
## 예제 3
### 입력
```
2 3
```
### 출력
```
0.666666667
```
